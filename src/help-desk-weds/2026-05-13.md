## Employee Adds a Problem

### Problem with Unsupported Software
```http
POST https://localhost:1337/employee/problems
Authorization: token joe@company.com
Content-Type: application/json

{
    "softwareId": "5dbe643d-37e0-44f2-bc51-a71fbeb5d0b4",
    "description": "Bungie hasn't updated Destiny in months.",
    "impact": "WorkStoppage"
}
```
